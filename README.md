# Global Supply Chain Orchestrator

Global Supply Chain Orchestrator is a Java-based application designed to manage, track, and optimize supply chain operations. 

## Project Structure and Navigation

If you are new to the project, here is how the code is organized so you can easily navigate it:

* `pom.xml` - This is the core Maven configuration file. It lists all the dependencies (libraries) your project uses, the Java version, and build plugins.
* `src/` - **This is where all the actual code lives.**
* `src/main/java/` - Contains all the application source code (Controllers, Services, Models/DTOs, Repositories).
* `src/main/resources/` - Contains application configuration files (like `application.properties` or `application.yml`) and static assets.

## Built With

* [Java](https://www.java.com/) - The programming language used.
* [Maven](https://maven.apache.org/) - Dependency Management.
